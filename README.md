# siakad3inf

SIAKAD (Sistem Informasi Akademik) adalah aplikasi pengelolaan data akademik yang mencakup mahasiswa, dosen, dan admin. Sistem ini dirancang untuk mempermudah proses administrasi kampus melalui pengelolaan data terpusat.

Aplikasi ini menyediakan fitur autentikasi untuk memisahkan akses berdasarkan role sehingga setiap user hanya dapat mengakses halaman yang sesuai dengan levelnya, dengan kredensial default sebagai berikut:

--Mahasiswa--
Username: mhs
Password: mhs

--Dosen--
Username: dosen
Password: dosen

--Admin--
Username: admin
Password: admin
